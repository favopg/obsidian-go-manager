You can use the existing plugin DataView to display a list of game records (SGF) in Vault.
You can search for players and sort by player.
You can create notes from SGF and save them as editable files using the existing plugin Go Board Viewer.

### Dependent Plugins
- BRAT
  - Used to update the Go Board Viewer plugin
- Go Board Viewer
  - Opens SGF files in view mode and edit mode for review.
- Dataview
  - Displays a list of game records in a table.
  - Used to manipulate data, apply search criteria, and sort.
